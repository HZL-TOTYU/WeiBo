# WeiBo

## 仿写微博客户端

这是一个仿写微博客户端的Swift程序，基于Swift 3.0 开发

### 近期主要功能实现

- 新浪微博首页（未登录）
- 新浪微博登录功能（Oauth 2.0 授权）
- 首页微博内容图文混排
- 微博页面的图片浏览器功能
- 微博未登录情况下访客视图
- 微博发布功能
- 微博表情键盘

## 安装和使用

- 直接clone/download到本地
- 打开终端到项目路径下执行 'pod update' 安装Pod依赖，(默认你CocoaPod已经装好)
- 打开Xcode运行程序
